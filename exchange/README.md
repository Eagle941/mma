# Exchange

This library keeps the connection with the exchange. It handles two types of
data:

- WebSocket data for order book updates
- REST requests for order management (create, edit, cancel)

This library supports the following exchanges:

- [ ] Replay from csv files
- [ ] Bybit exchange spot
- [ ] Binance exchange spot

When connecting to crypto exchanges, it supports switching between testnet and
live trading.
